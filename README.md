## Technology

## Local Development
Run `yarn` and `yarn start`.

## How to get GitHub Token
* Settings -> Developer Settings -> Personal access tokens -> Generate new token
* Check `repo` and `user`
  * Current version doesn't provide the repository search function. It'll be coming soom.
* Generate token
